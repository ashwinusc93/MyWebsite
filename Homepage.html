Wireless Smart Energy Management System

In this project we automate the usage of devices with the help of a rules engine based on energy usage or time of use. The idea of this project was to build a system where we raise energy awareness through automation of devices by storing energy data in the cloud. This will be very useful in developing countries like India where the power grid is still not smart enough to regulate the usage of power and overcome wastage of power, and there are chronic power shortages. So in this project the energy data of the devices is stored in the Azure cloud. The data is first collected by an intermediate C# server which deserialises the XML data sent by the wireless sensors, which is then sent to the Azure database(Table Storage). The device is then regulated based on the energy data usage or time of use. An ASP.NET based web client was developed where the energy data usage of the devices is displayed to the user, where the user can choose the time where he wants to find the energy usage, so that he can use the knowledge provided by the data graphs in the website and make appropriate use of energy in the future. The project has another facet where events like gas leaks or burglary alarms are sent to the Azure Event Hub, the event is processed and the appropriate action is taken based on the event processing
